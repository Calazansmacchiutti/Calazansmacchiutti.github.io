Project Overview

This project implements a comprehensive fraud detection system combining Know Your Customer (KYC) and Know Your Transaction (KYT) methodologies. Using the Credit Card Fraud Detection dataset, the system evaluates multiple machine learning approaches including supervised classifiers, unsupervised anomaly detection, and deep learning autoencoders to identify fraudulent transactions in highly imbalanced datasets.

Key Questions Answered

  • How do supervised models (XGBoost, Random Forest) compare against unsupervised approaches (Isolation Forest, K-Means, Autoencoders) for fraud detection?
  • Which PCA-transformed features show the strongest discriminative power between normal and fraudulent transactions?
  • What is the optimal threshold strategy when dealing with extreme class imbalance (~0.17% fraud rate)?
  • How can autoencoder reconstruction error be leveraged for anomaly-based fraud detection?

Analytical Highlights

  • Comprehensive model comparison: XGBoost achieved highest Average Precision (0.858) followed by Random Forest (0.826)
  • ROC-AUC scores above 0.96 for all models, with XGBoost reaching 0.969
  • Autoencoder-based anomaly detection with tuned threshold achieving AP of 0.510
  • Feature importance analysis revealing V14, V4, V12, and V10 as top discriminators
  • Transaction amount analysis showing frauds concentrated in lower value ranges

Methodology

  1. Data exploration and visualization of PCA-transformed features by class
  2. Implementation of supervised classifiers: XGBoost and Random Forest with class balancing
  3. Unsupervised anomaly detection: Isolation Forest, K-Means clustering
  4. Deep learning approach: Autoencoder trained on normal transactions for reconstruction error-based detection
  5. Threshold optimization using precision-recall trade-offs
  6. Model evaluation with focus on Average Precision due to class imbalance

Results & Impact

The study demonstrates that supervised learning approaches significantly outperform unsupervised methods for fraud detection when labeled data is available. XGBoost emerged as the best performer with 85.8% Average Precision, capable of detecting 86% of fraudulent transactions while maintaining 51% precision. The autoencoder approach, while achieving lower precision, provides a viable alternative when labeled fraud examples are scarce, making it suitable for cold-start scenarios in new fraud detection deployments.
